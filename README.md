
## MERN todo-item App

### Description
* This application is a todo-item app. In this we can add an item and list all the items. It is possible to edit  as well as 
  delete the items. The full MERN CRUD operation is implemented here. 

### Technology Used
* Mongo DB
* Express
* NodeJS
* React
